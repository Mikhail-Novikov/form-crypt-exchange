# Проект: Форма обмена валюты с указанием реквизитов

## Описание

Интерфейс для управления банковскими реестрами

### Функции

- Добавление банковских реестров
- Редактирование банковских реестров
- Удаление банковских реестров

### Технологии

- React
- TypeScript
- Material-UI

### Установка

1. Клонируйте репозиторий
2. Запустите `npm start`
